# 文档处理工作台 - 用户使用指南

## 一、系统概述

文档处理工作台是一个基于浏览器的在线文档处理平台，无需安装任何软件，通过浏览器即可完成以下任务：

| 功能模块 | 支持格式 | 输出格式 | 说明 |
|---------|---------|---------|------|
| 证件 OCR 翻译 | JPG、PNG、BMP、TIF | 图片 | 身份证、结婚证等证件图片识别翻译 |
| 驾驶证翻译 | 图片 | DOCX | 单图处理、多图合并、批量输出 |
| 数字专检 | DOCX | DOCX | 原文与译文数字一致性检查 |
| 中翻译专检 | DOCX、PDF | DOCX | 原文译文对比与自动修复 |
| 多语对照记忆 | DOCX | XLSX | 段落对齐生成语料库 |
| 通用证件翻译 | PDF、图片 | DOCX | 多语种翻译（一对多） |
| 文档预处理 V2 | PDF、图片 | DOCX | 扫描件、票据、表格类文档 OCR |

---

## 二、访问方式

### 启动服务

```bash
# 在项目根目录执行
python app/main.py
```

服务启动后，访问 **http://localhost:8001**

### 页面导航

- **首页** (`/`)：所有功能入口导航
- **任务工作台** (`/dashboard`)：查看所有任务状态、进度和历史记录

## 三、各功能模块使用说明

### 3.1 证件 OCR 翻译

**入口**：`首页` → `证件 OCR 翻译` 或直接访问 `/ocr`

**适用场景**：身份证、结婚证等证件的图片识别与翻译

**操作步骤**：

1. **上传图片**
   - 将证件图片拖拽到上传区域，或点击选择文件
   - 支持 JPG、PNG、BMP、TIF 格式，最大 50MB

2. **配置参数**
   - **证件类型**：身份证 / 结婚证
   - **源语言**：中文 / 英文 / 日文 / 韩文
   - **目标语言**：中文 / 英文 / 日文 / 韩文
   - **证件面**（仅身份证）：正面（头像面）/ 背面（国徽面）
   - **生成 OCR 可视化图**：勾选可在结果中查看识别区域标注

3. **提交处理**
   - 点击「开始 OCR 翻译」按钮
   - 等待处理完成（可在任务工作台查看进度）

4. **下载结果**
   - 处理完成后，页面显示翻译结果图片
   - 点击「下载翻译结果图」保存

---

### 3.2 驾驶证翻译

**入口**：`首页` → `驾驶证翻译` 或直接访问 `/drivers-license`

**适用场景**：驾驶证首页、副页、图例页等单次合并或批量翻译

**操作步骤**：

1. **上传图片**
   - 拖拽或点击上传驾驶证图片
   - 支持多张图片同时上传

2. **选择处理模式**
   - **单图模式**：每张图片生成独立译文
   - **合并模式**：多张图片合并为一份 Word 文档
   - **批量模式**：批量处理，分别输出

3. **提交处理**
   - 点击「开始翻译」按钮

4. **下载结果**
   - 点击对应的下载按钮获取 DOCX 文件

---

### 3.3 数字专检

**入口**：`首页` → `数字专检` 或直接访问 `/number-check`

**适用场景**：交付前核查原文与译文的数字是否一致

**操作步骤**：

1. **上传文件**
   - 上传**原文** DOCX 文件
   - 上传**译文** DOCX 文件

2. **开始检查**
   - 点击「开始数字专检」按钮

3. **查看报告**
   - 系统自动对比两份文档中的数字
   - 显示差异数量和具体位置

4. **下载修复结果**
   - 可下载经过批注的 DOCX 文件

---

### 3.4 中翻译专检

**入口**：`首页` → `中翻译专检` 或直接访问 `/zhongfanyi`

**适用场景**：原文与译文的深度审校，支持规则文件和 AI 辅助检查

**操作步骤**：

1. **上传文件**
   - 上传**原文**文件（支持 DOCX、PDF、DOC）
   - 上传**译文**文件（支持 DOCX、PDF、DOC）

2. **（可选）配置规则**
   - 上传规则文件（PDF/DOCX/TXT）
   - 勾选「使用 AI 规则」让 AI 自动总结规则

3. **开始检查**
   - 点击「开始中翻译专检」按钮

4. **查看与下载**
   - 查看差异报告
   - 下载批注后的 DOCX 文件

---

### 3.5 多语对照记忆

**入口**：`首页` → `多语对照记忆` 或直接访问 `/alignment`

**适用场景**：生成原文与译文的对照语料，适合术语整理和翻译记忆库沉淀

**操作步骤**：

1. **上传文件**
   - 上传**原文** DOCX 文件
   - 上传**译文** DOCX 文件

2. **设置阈值参数**（可选）
   - 相似度阈值：控制段落匹配灵敏度
   - 缓冲区大小：调整匹配范围

3. **开始处理**
   - 点击「开始对照」按钮

4. **下载结果**
   - 生成 Excel 格式对照表
   - 每行包含序号、原文、译文、相似度等信息

---

### 3.6 通用证件翻译

**入口**：`首页` → `通用证件翻译` 或直接访问 `/doc-translate`

**适用场景**：PDF 或图片的通用翻译，支持多种目标语言

**操作步骤**：

1. **上传文件**
   - 拖拽或点击上传 PDF 或图片
   - 支持 PDF、PNG、JPG 等格式

2. **选择目标语言**
   - 点击语言标签选择目标语言
   - 支持多选（一次翻译成多种语言）

3. **提交处理**
   - 点击「开始翻译」按钮

4. **下载结果**
   - 每个语种生成一个 DOCX 文件
   - 点击对应按钮下载

---

### 3.7 文档预处理 V2

**入口**：`首页` → `文档预处理V2` 或直接访问 `/pdf2docx`

**适用场景**：将扫描件、PDF、图片等不可编辑文档转换为可编辑的 Word 文档

**操作步骤**：

1. **上传文件**
   - 拖拽或点击上传 PDF 或图片
   - 支持 PDF、PNG、JPG、JPEG、BMP、GIF、WEBP

2. **选择转换模型**
   - **快速版 V2**：处理速度快，适合清晰简单的文档
   - **增强版 V2**：识别效果更好，适合复杂版式、表格

3. **提交处理**
   - 点击「开始转换」按钮

4. **下载结果**
   - 生成 DOCX 可编辑文档
   - 同时提供原始 OCR 文本供复核

---

## 四、任务工作台使用指南

访问 `/dashboard` 进入任务工作台

### 4.1 查看任务统计

页面顶部显示各类任务的数量统计：

- **全部任务**：所有任务总数
- **排队中**：等待处理的任务
- **处理中**：正在执行的任务
- **已完成**：成功完成的任务
- **失败**：执行失败的任务
- **已取消**：被取消的任务

### 4.2 筛选任务

- **按状态筛选**：点击顶部统计卡片，只显示对应状态的任务
- **按类型筛选**：使用下拉框按功能模块筛选（OCR翻译、数字专检等）
- **关键词搜索**：在搜索框输入文件名搜索

### 4.3 查看任务详情

点击任务操作栏的「详情」按钮，可查看：

- 任务编号和类型
- 提交时间和耗时
- 处理状态和进度
- 处理日志
- 输入/输出文件列表

### 4.4 下载和取消

- **下载**：在详情页或任务列表下载已完成的输出文件
- **取消**：对排队中或处理中的任务，可点击「取消」终止执行

## 五、常见问题

### 文件上传失败

- 检查文件格式是否符合要求
- 确认文件大小在限制范围内（最大 50MB）
- 尝试刷新页面后重新上传

### 任务一直处于排队中

- 系统可能正在处理其他任务，请稍候
- 可在任务工作台查看总体负载情况

### 处理失败怎么办

- 点击任务详情报错信息
- 根据错误提示调整参数或文件后重试
- 可联系技术支持获取帮助

### 支持的文件格式

| 功能 | 支持输入格式 | 输出格式 |
|-----|------------|---------|
| OCR翻译 | JPG, PNG, BMP, TIF | 图片 |
| 驾驶证翻译 | 图片 | DOCX |
| 数字专检 | DOCX | DOCX |
| 中翻译专检 | DOCX, PDF, DOC | DOCX |
| 对照记忆 | DOCX | XLSX |
| 通用证件翻译 | PDF, 图片 | DOCX |
| 文档预处理 | PDF, 图片 | DOCX |

## 六、技术支持

如遇问题，请检查：

1. 服务是否正常运行（无报错退出）
2. 网络连接是否正常
3. 文件格式和大小是否符合要求

如需进一步帮助，请联系系统管理员。

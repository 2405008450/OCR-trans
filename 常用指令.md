常用管理命令：
构建
sudo docker-compose up -d --build

# 查看运行状态
docker compose ps

# 查看日志
docker compose logs -f

# 停止
docker compose down
注意： 启动前需要先在项目根目录创建 .env 文件并填入真实的 API Key：


copy env.example .env
